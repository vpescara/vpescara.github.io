Creación de Molde.

Para crear un molde vamos a tener que seguir los siguientes pasos:

1- Crear objeto final.

La forma más fácil de crear un molde consiste en la creación digital del objeto final que deseamos, de este modo, partiremos generando el diseño base de nuestro objeto, en este caso la construcción de un cenicero.

Partiremos diseñando el perfil de uno de los lados de nuestro cenicero:

Luego, utilizaremos Revolve para darle volumen a la figura de manera pareja. Gracias a esto ya tendremos el modelo base el cual podremos modificar y editar a gusto.

Seguiremos utilizando el comando Shell en la cara superior para darle la profundidad que necesitara.

Luego podemos agregar detalles, en este caso partiremos por agregar el cilindro del medio que se utiliza para apagar los cigarros. Para ello utilizaremos el comando Cilindro y lo colocaremos en el centro de nuestro cenicero, las dimensiones pueden ser a gusto.

Y finalizaremos con un FilletEdge para redondear un poco los bordes y que sea más cómodo tanto para el manejo postproducción y la facilidad de que el relleno llegue a todos lados del molde.

2- Crear Molde.

Ahora, para la creación de nuestro molde en primera instancia necesitamos identificar como podría funcionar. En este caso utilizaremos un molde simple de 2 partes, una inferior que servirá para hacer la forma exterior del cenicero y una superior que nos proporcionará los detalles interiores.

Partiremos con la parte inferior:

Iniciaremos con un cuadrado en Sketch el cual debe ser un poco más grande que el tamaño de nuestro objeto a modo de poder contenerlo.

Luego usando Extrude le daremos volumen a nuestro cuadrado a modo que sobrepase el cenicero.

Una vez la base inferior este lista, tendremos que posicionar nuestro objeto en el borde de nuestro molde, tal y como muestra la siguiente imagen.

Esto se hace a modo de poder dividir el espacio interior y el exterior de nuestro cenicero.

Ahora, utilizaremos el comando Combine seleccionando el “cuerpo de destino” lo que será nuestro molde y como “cuerpo de herramienta” nuestro objeto. Luego utilizaremos la opción Cut para generar los relieves que necesitamos.

Importante escoger la opción de “Conservar Herramienta” ya que de otra manera perderemos nuestro objeto final.

Una vez teniendo nuestras dos partes separe la interior hacia arriba para poder trabajar más cómodamente en ella.

Parte superior:

Hasta el momento ya tenemos nuestro molde inferior terminado, ahora finalizaremos el superior de manera muy sencilla.

Para ello, al igual que lo hicimos en un principio generaremos un cuadrado mayor que el objeto (recomiendo del mismo tamaño que el que se realizó al inicio), luego generaremos nuestro solido con Extrude y usando Combine lo juntaremos con nuestra parte interior.

Para este caso es necesario escoger la opción “Unir” a la hora de realizar el Combine.

Al finalizar uno deberia quedarse con 3 objetos finales:

  • El cenicero que modelo en un principio.
  • El molde inferior.
  • El molde superior.

Exportar para imprimir.

Finalmente, para exportar nuestros moldes para ser impresos, nos iremos a la ventana de “Herramientas” o “Tools” y escogeremos Impresión en 3D.

Para la configuración recomiendo el formato STL, “Refinado” en Alto y en “Salida” dejar en blanco para que le pida ruta de guardado al finalizar.
